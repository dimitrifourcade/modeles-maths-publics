# Règles IA globales

Ce dossier contient les règles pédagogiques, stylistiques et rédactionnelles globales du projet.

Le fichier principal est :

```text
regles_ia_globales.json
```

Il est lu par la page `/generer-avec-modeles` pour enrichir automatiquement le bloc IA transmis à ChatGPT.

Rôle de `index.json` : décrire les modèles.
Rôle de `regles_ia_globales.json` : imposer les règles communes à tous les documents générés.
