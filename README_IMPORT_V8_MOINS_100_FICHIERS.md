# Import GitHub v8 — essentiel, moins de 100 fichiers

Cette version est prévue pour l'upload web GitHub.

Elle contient uniquement les fichiers utiles à ajouter : PDF, source TeX, manifestes, JSON et rapports.
Les nombreux `README.md` de squelette ont été retirés pour passer sous la limite GitHub de 100 fichiers.

## À importer

Depuis GitHub, à la racine du dépôt `modeles-maths-publics` :

1. `Add file` → `Upload files`.
2. Dézipper cette archive sur l'ordinateur.
3. Ouvrir le dossier `upload_v8_essentiel`.
4. Glisser dans GitHub le contenu du dossier, notamment :
   - `documents_a_classer/`
   - `README_IMPORT_V8_MOINS_100_FICHIERS.md`
5. Vérifier que les fichiers s'ajoutent dans `documents_a_classer/...`.
6. Commit.

Ne pas glisser le dossier parent `upload_v8_essentiel` lui-même.
