# Feuille-méthode — Point fixe — Terminale spécialité — v4 validée

Ce dossier contient le **vrai modèle validé** de feuille-méthode sur le point fixe en Terminale spécialité.

## Fichiers

- `fm_point_fixe_tspe_v4.tex` : source LaTeX de la feuille élève/modèle.
- `fm_point_fixe_tspe_v4.pdf` : PDF compilé de la feuille.
- `corr_fm_point_fixe_tspe_v4.tex` : source LaTeX de la correction séparée.
- `corr_fm_point_fixe_tspe_v4.pdf` : PDF compilé de la correction séparée.
- `apercu_fm_point_fixe_tspe_v4_page1.png` : aperçu de la page 1 de la feuille.
- `apercu_corr_fm_point_fixe_tspe_v4_page1.png` : aperçu de la page 1 de la correction.

## Statut

Modèle validé prioritaire pour les futures feuilles-méthodes.

## Règles de style à reprendre

- Style sobre, presque sans couleur.
- Mise en page aérée.
- Encadrés clairs, non tassés.
- Correction séparée.
- L'exemple corrigé doit reprendre exactement la rédaction de l'encadré méthode.
- Ne pas laisser d'espace élève inutile.
- Mieux vaut passer à deux pages que tasser le document.

## Usage IA

Lorsqu'un document est généré à partir de ce modèle, l'IA doit conserver la logique suivante :

1. objectif clair ;
2. méthode en étapes ;
3. pièges fréquents ;
4. rédaction type ;
5. exemple corrigé complet suivant exactement la rédaction modèle ;
6. exemple guidé si pertinent ;
7. correction séparée.
