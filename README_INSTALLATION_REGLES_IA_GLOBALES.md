# Installation — règles IA globales

Uploader le contenu de cette archive à la racine du dépôt GitHub `modeles-maths-publics`.

Cela crée :

```text
regles_ia/regles_ia_globales.json
regles_ia/README.md
regles_ia/SCHEMA_REGLES_IA_GLOBALES.md
```

Après commit, le site utilisera automatiquement :

```text
https://raw.githubusercontent.com/dimitrifourcade/modeles-maths-publics/main/regles_ia/regles_ia_globales.json
```

Si ce fichier n’est pas présent, le site garde une copie locale de secours.
