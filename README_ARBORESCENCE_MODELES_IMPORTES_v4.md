# Arborescence v4 — modèles importés par niveau et type

Arborescence destinée au dépôt GitHub public `modeles-maths-publics`.

Principe : tout document déposé ici est déjà considéré comme un modèle ou une référence exploitable. Cette zone sert à ranger proprement les fichiers importés avant, ou pendant, leur référencement dans `index.json`.

Les dossiers contiennent un `README.md` pour être visibles dans GitHub, même avant l'ajout de fichiers.
