# Documents extraits d’archives

Ce dossier est alimenté automatiquement par la GitHub Action `.github/workflows/extract_archives.yml`.

Principe :

1. déposer une archive `.zip` dans `documents_a_classer/` ou l’un de ses sous-dossiers ;
2. GitHub Actions extrait les fichiers utiles ;
3. les PDF, TeX, images et fichiers texte sont publiés ici ;
4. `index_archives_extraites.json` est mis à jour à la racine du dépôt ;
5. le site peut alors les afficher comme des modèles importés ordinaires.

Ne pas modifier manuellement ce dossier sauf besoin exceptionnel.
