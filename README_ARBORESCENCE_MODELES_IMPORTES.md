# Arborescence GitHub — modèles importés

Cette archive crée une structure visible dans GitHub pour ranger les documents déposés dans `documents_a_classer/`.

Contrairement à une version avec `.gitkeep`, chaque dossier contient ici un `README.md`, donc les sous-dossiers apparaissent clairement dans l’interface GitHub.

## Principe

Tous les fichiers déposés sont considérés comme des modèles ou références exploitables. La page du site `/modeles-importes` permet ensuite de générer une entrée `index.json` pour les référencer officiellement dans `/modeles`.
