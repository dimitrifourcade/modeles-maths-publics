# Guide de rangement rapide

## Dépôt rapide
Mettre un fichier dans `00_inbox_a_trier/` quand le rangement exact n'est pas encore choisi.

## Rangement conseillé
Pour un modèle lycée, utiliser :

```text
documents_a_classer/01_lycee/<niveau>/<type_document>/
```

Exemple :

```text
documents_a_classer/01_lycee/terminale_specialite/03_fiches_methodes/
```

## Référencement officiel
Une fois le modèle prêt à apparaître dans `/modeles`, générer ou compléter une entrée dans `index.json`.

Le dossier `90_modeles_a_referencer_index_json/` sert uniquement de sas pour les modèles déjà retenus mais pas encore déclarés dans `index.json`.
